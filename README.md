# 🧠 Mental Age Test – AI Personality Analyzer

A production-grade, mobile-first Progressive Web App (PWA) built with **React 18**, **TypeScript**, and **OpenRouter AI**.

## 🚀 Key Features
- **AI Mental Age Scoring**: Advanced logic weighted by category-specific algorithms.
- **Personality Archetypes**: Personalized summaries, strengths, and advice generated by AI.
- **PWA Experience**: Installable on home screen, works offline, and features mobile-native navigation.
- **Glassmorphism UI**: Premium "Dark Navy & Neon" design system using Styled Components.
- **Multi-language Support**: Fully localized in English, Hindi, Spanish, French, German, and Italian.
- **Social Sharing**: High-resolution image generation and Web Share API integration.

## 🛠️ Tech Stack
- **Framework**: React 18 + Vite
- **Logic**: TypeScript 5
- **Styling**: Styled Components + Framer Motion
- **AI**: OpenRouter (Google Gemma 3 / Liquid LFM)
- **Persistence**: LocalStorage / IndexedDB
- **Sharing**: html2canvas + Web Share API

## 📦 Installation
1. Clone the repository.
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and add your `VITE_OPENROUTER_API_KEY`.
4. Start development server: `npm run dev`

## 💰 Monetization
Supports Banner, Native, and Rewarded Ads via the integrated AdsContext system.
